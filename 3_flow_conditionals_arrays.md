# Ruby Flashcards

## Flashcard 1  
**Q:**  
- How do you structure a basic if statement in Ruby?

**A:**  
```ruby
if condition
  # code when condition is truthy
end
```

---

## Flashcard 2
**Q:**  
- What is the Ruby ternary operator?

**A:**
```ruby
condition ? code_when_truthy : code_when_falsey
```
